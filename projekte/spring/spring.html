Spring Framework

Ein prototypischer Webshop wurde mit dem Spring Framework, Thymeleaf und Bootstrap erstellt. Mit der HTTP Get Methode werden die unterschiedlichen Seiten auf der Webseite angezeigt und durch Thymeleaf serverseitig in das Bootstrap Frontend gerendert. Ein Websocket wird für die Chat-Kommunikation genutzt. Dabei kommunizieren zwei Clients miteinander. Der Client kann einen Artikel in den Warenkorb legen. Dies wird durch die HTTP Post Methode und JavaScript realisiert.
